Construir FIS tipo 2

Convierta el FIS de tipo 1 en un FIS fis1de tipo 2.
fis2 = convertToType2 (fis1);
El sistema Sugeno de tipo 2 fis2utiliza funciones de pertenencia de tipo 2 para las variables de entrada y funciones de pertenencia de tipo 1 para las variables de salida.
Defina la huella de incertidumbre (FOU) para los MF de entrada como se define en [1]. Para ello, establezca el factor de escala de MF más bajo para cada MF. Para este ejemplo, establezca los valores de retardo de MF más bajos en 0.
scale = [0.2 0.9 0.2;0.3 0.9 0.3];
for i = 1:length(fis2.Inputs)
for j = 1:length(fis2.Inputs(i).MembershipFunctions)
fis2.Inputs(i).MembershipFunctions(j).LowerLag = 0;
fis2.Inputs(i).MembershipFunctions(j).LowerScale = scale(i,j);
end
end
