This spline-based curve is a mapping on the vector x, and is named because of
its S-shape.

The sigmoid curve plotted for the vector x depends on two parameters a and c as
given by

y = psigmf(x,[a1 c1 a2 c2])

function []=FunComple(DomA,DomB,a,b,c,Comp)
clc
close all
x=linspace(DomA,DomB,100);
n=size(x);
A=zeros(1,n(2));
for i=1:n(2)
if x(i)<=a
A(i)=0;
elseif (a<x(i)) && (x(i)<=b)
A(i)=(x(i)-a)/(b-a);
elseif (b<x(i)) && (x(i)<c)
A(i)=(c-x(i))/(c-b);
else
A(i)=0;
end
end
B=zeros(1,n(2));
for i=1:n(2)
if Comp==1
B(i)=1-A(i);
elseif Comp==2
if i==1
S=input('Introdusca el par√°metro s de sugeno(s>-1): ');
end
B(i)=(1-A(i))/(1+S*A(i));
else
if i==1
w=input('Introdusca el par√°metro w de Yager(w>0): ');
end
B(i)=(1-A(i)^w)^(1/w);
end
end
hold on
plot(x,A,'Color',[0 0.55 0],'LineWidth',3);
plot(x,B,'Color',[0.55 0 0],'LineWidth',3);
grid on
axis([DomA-1,DomB+1,-0.1,5.1])
title('Gr√°fica del conjunto difuzo y complemento')
h = gca; % ejes actuales
h.XAxisLocation = "origin"; % eje en el origen
h.YAxisLocation = "origin"; % eje en el origen
h.Box = "off";
end
